# TC Frittlingen e.V. – Website & Padel-Buchung

Vereinswebsite des Tennisclub Frittlingen e.V. mit integriertem Padel-Buchungssystem.

## Inhalt

- **`MockUp Padel.html`** – Die komplette Single-Page-Website (Startseite, Tennis, Verein, Spielbetrieb, Mannschaften, Aktuell, Padel-Buchung).
- Bild- und PDF-Dateien (Plätze, Mannschaften, Anfahrt, Downloads).

## Features

- Responsives Design für Handy & Desktop (Tailwind CSS)
- Mobile-Navigation (Hamburger-Drawer) & Desktop-Navigation
- **Padel-Buchungssystem** mit
  - Datumsauswahl (14 Tage)
  - Belegungsraster im 30-Minuten-Takt für 2 Courts
  - dynamischer Preisberechnung (Mitglied 3,75 €/h · Gast 7,50 €/h pro Anteil)
  - **Schlägerverleih** (2,00 € pro Schläger, 0–4 Schläger wählbar)
  - Buchungs-Modal mit Spielereingabe, Zahlungsauswahl & Bestätigung
- KI-Chat-Assistent (Demo) für Desktop-Sidebar & Mobile-Bottom-Sheet
- Google-Maps-Anfahrt

## Nutzung

Die Website ist eine eigenständige HTML-Datei. Zum Ansehen einfach `MockUp Padel.html` im Browser öffnen.
